# Task 2 – Student Digital Portfolio

## Platforms Set Up

### 1. GitHub (github.com)
A platform for hosting code and version-controlled projects. Used by developers, students, and researchers worldwide to collaborate and showcase their work. Created a profile README repository with name, branch, year, and learning goals.

### 2. LinkedIn (linkedin.com)
A professional networking platform used by employers, recruiters, and industry professionals. Filled in the Education section with B.Tech CSE (AI-ML), VIT Bhopal University, and expected graduation year (2029).

### 3. Kaggle (kaggle.com)
A data science and machine learning platform where students and professionals can participate in competitions, access datasets, and complete learning courses. Especially relevant for an AI-ML student.
